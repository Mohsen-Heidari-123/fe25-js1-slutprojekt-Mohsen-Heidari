# Movie App – TMDB API (The Movie Database)

This is a web project that displays movies, TV shows, and people using the TMDB API.  
The project fetches different types of information and details, and TMDB provides many APIs that can be tested and explored.

## Features
- Display popular and top-rated movies
- Display TV shows
- Display actors and other people
- Search for movies or people
- Animations using GSAP library
- Uses JavaScript modules with import and export, Uses person images and movie images from the API

## Technologies
- HTML
- CSS
- JavaScript
- GSAP (animation library)
- TMDB API
- JavaScript Modules

## Structure
- `index.html` – home page
- `movies.html` – top-rated and popular movies page
- `person.html` – people page
- `js/` – JavaScript files
- `css/` – CSS files

## Project Status
This project was created as a final project and is now completed.  
It was a good introduction to building a web project where data can be fetched and tested using external APIs.
